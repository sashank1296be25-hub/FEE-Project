# Fluid Ledger 💸

Fluid Ledger is a lightweight, privacy-first personal finance application that operates entirely offline. Built with a strict "Zero-Cloud" architecture, it ensures that your sensitive financial data never leaves your personal device. 

Instead of relying on third-party servers, Fluid Ledger uses your browser's native capabilities to securely store and calculate your daily income and expenses.

## ✨ Key Features

* **100% Offline & Private:** No servers, no external databases. All transaction data is saved strictly to your browser's LocalStorage.
* **Real-Time Analytics:** Instantly calculates and updates your total balance, total income, and total expenses with zero network latency.
* **Lightweight & Fast:** Built entirely using fundamental web technologies without any heavy or bloated frontend frameworks.
* **Responsive UI:** A clean, intuitive, and mobile-friendly design using CSS Flexbox, featuring color-coded visual cues for incoming and outgoing funds.

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Flexbox)
* **Logic & DOM Manipulation:** Vanilla JavaScript
* **Data Persistence:** Browser LocalStorage API
